CSS Sprites - Irregular shapes 1

The big difference from the previous page is where we are positioning the <li> tags.
Notice that the links are neither aligned horizontally nor vertically, but rather in specific positions.
The goal here is to surround each graphical element with a box that surrounds each of the shapes.

The CSS involved

     
    #shapes {
     width: 300px;
     height: 150px;
     background: url(../images/shapes.gif);
     margin: 10px auto; padding: 0;
     position: relative;
    }
    
    #shapes li {margin: 0; padding: 0; list-style: none; display: block; position: absolute;}

    #shapes a {display: block; border: solid 1px #f00;}

    #shape1 {left: 9px; top: 7px; width: 86px; height: 130px;}
    #shape2 {left: 77px; top: 16px; width: 79px; height: 86px;}
    #shape3 {left: 160px; top: 0px; width: 112px; height: 77px;}
    #shape4 {left: 173px; top: 57px; width: 120px; height: 80px;}
    #shape5 {left: 110px; top: 102px; width: 98px; height: 45px;}
    #shape1 a {height: 130px;}
    #shape2 a {height: 86px;}
    #shape3 a {height: 77px;}
    #shape4 a {height: 80px;}
    #shape5 a {height: 45px;}
