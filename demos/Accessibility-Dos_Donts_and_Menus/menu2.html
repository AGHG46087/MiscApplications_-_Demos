Page Navigation

Page Navigation is simple with a maenu such as this, where the user can either use the mouse or the keyboard. It is important to note: that this solution is driven by 

<ul>,<li>,<a>, CSS and Javascript
controlling the behavior.

Keyboard Navigation

The page top navigation can easily be traversed on this nav, by the use of the keyboard only. In this case simply hitting the tab key will take the user to each tab stop.

Mouse Navigation

The page not only being accessible by the keyboard it is also accessible by the use of the mouse, in this case we have achieved device independence.
