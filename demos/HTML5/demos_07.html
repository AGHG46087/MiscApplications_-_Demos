This page demonstrates the markup used in a typical document to support the W3C Aria Role specification.

  • Through the use of the role attribute we can indicate landmark elements in the document - such as page headings, navigation, search and main content.
  • The Juice Studio Accessibility Toolbar is a free plugin for Firefox that will provide a visual display of the landmark roles currently defined in a document/webpage.
  • These Landmark Roles allows Assistive Technology users to quickly navigate to different areas of the document. For example on JAWS CTRL + INSERT + ; will open the Landmarks dialog.
So What
  • ARIA aimed at solving dynamically updated pages and provide Accessibility for RIA
  • ARIA, combined with CSS selectors provides for both Visual presentation and Accessibility
  • ARIA Roles are supported in JAWS, NVDA, VoiceOver
