CSS Radial Gradients

background: radial-gradient( , circle , #0000FF, #0000FF);
background: gradient(radial, , 0, 50% 50%, , from(#0000FF), to(#0000FF));
So What
  • Can support a fallback for browsers that do not support using an image
  • If the browser supports Gradients, then image is not downloaded, less HTTP requests, less packets, less header traffic
div { background-color: #1a82f7; /* fallback color */ background-image: url(images/radial_bg_2.png); /* fallback image */ background-image: -moz-radial-gradient(100% 100% 90deg, #2F2727, #1a82f7); background-image: -webkit-gradient(radial, 0% 0%, 0% 100%, from(#1a82f7), to(#2F2727)); }
