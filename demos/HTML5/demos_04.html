CSS Linear Gradients

background: linear-gradient( , #0000FF, #0000FF);
background: gradient(linear, , , from(#0000FF), to(#0000FF));
So What
  • Can support a fallback for browsers that do not support using an image
  • If the browser supports Gradients, then image is not downloaded, less HTTP requests, less packets, less header traffic
div { background-color: #1a82f7; /* fallback color */ background-image: url(images/linear_bg_2.png); /* fallback image */ background-image: -moz-linear-gradient(100% 100% 90deg, #2F2727, #1a82f7); background-image: -webkit-gradient(linear, 0% 0%, 0% 100%, from(#1a82f7), to(#2F2727)); }
