Color Contrast Luminance

Guideline 1.4 of WCAG 2 states, make it easy to distinguish foreground information from background images or sounds.

The luminosity contrast ratio algorithm is the recommendation WCAG 2. Brightness is measured by relative luminance(L) where L is calculated by:

            L = 0.2126 * R + 0.7152 * G + 0.0722 * B where R, G and B are defined as:
            if RsRGB <= 0.03928 then R = RsRGB/12.92 else R = ((RsRGB+0.055)/1.055) ^ 2.4
            if GsRGB <= 0.03928 then G = GsRGB/12.92 else G = ((GsRGB+0.055)/1.055) ^ 2.4
            if BsRGB <= 0.03928 then B = BsRGB/12.92 else B = ((BsRGB+0.055)/1.055) ^ 2.4
               and RsRGB, GsRGB, and BsRGB are defined as:
                   RsRGB = R8bit/255
                   GsRGB = G8bit/255
                   BsRGB = B8bit/255
            The calculation (L1 + 0.05) / (L2 + 0.05) should result in a number greater than 4.5:1.
The Result of the formula is measured against the font size and weight that determines the actual PASS/FAIL.

WCAG 2. 1.4.3: Contrast Minimum (Level AA)

  • General Text and images of text have a contrast ratio of at least 4.5:1.
  • Text Less than 18pt font normal weight (less than 14pt regardless of weight) and color contrast ratio of at least 4.5:1.
  • Large text between 14pt and 18pt font bold and has a color contrast ratio of at least 3:1.
  • Large text greater than 18pt font rgardless of weight and has a color contrast ratio of at least 3:1.

WCAG 2. 1.4.6: Contrast Enhanced (Level AAA)

  • General Text and images of text have a contrast ratio of at least 7:1.
  • Text Less than 18pt font normal weight (less than 14pt regardless of weight) and color contrast ratio of at least 7:1.
  • Large text between 14pt and 18pt font bold and has a color contrast ratio of at least 4.5:1.
  • Large text greater than 18pt font rgardless of weight and has a color contrast ratio of at least 4.5:1.

Color Brightness / Difference (AERT-WCAG 1)

Guideline of WCAG 1 does not in fact, provide any specific guidelines concerning color contrast. The formulas commonly used to judge brighness and differnece were specified in Techniques For Accessibility Evaluation And Repair Tools (AERT).

The AERT provides two formulas to assist, the issue is that they are independent and one does not account for the other. They are provided to help authors conform to the WCAG 1 but are not actually part of the WCAG 1 document:
Color Brightness

            ((Red value X 299) + (Green value X 587) + (Blue value X 114)) / 1000
Color Difference
            (maximum (Red value 1, Red value 2) - minimum (Red value 1, Red value 2)) +
            (maximum (Green value 1, Green value 2) - minimum (Green value 1, Green value 2)) +
            (maximum (Blue value 1, Blue value 2) - minimum (Blue value 1, Blue value 2))

Color challenges

For users that do not experience color contrast deficiencies, saturation and hue have little or no effect on legibility assessed through reading performance. Vision impairments that affect color contrast may affect luminance contrast to some degree. Hence the recommendation with respect to color contrast includes the calculation in such a way that color is not a key factor so that people who have color vision deficit will also have sufficient contrast between foreground and background

Text and images of text that is larger in size or contains wider character strokes is considered easier to read at lower contrast. The color contrast requirement for larger text is lower. Text that falls into this category is a font that is 18 point text or 14 point bold text. This allows for designers to use a wider range of color choices for large text that is consistent with titles and headers.

Images of text being an image are more difficult to change the foreground and background color contrast and color combinations for images of text, which would be required for some users to meet this requirement

NOTE: In addition, the enlargement of images of text also does not scale clearly during enlargement as the images tend to realize the pixilation in the colors chosen. Therefore, it is strongly recommended using text where ever possible; if the image of text cannot be rendered as standard text fonts, consider supplying an image of high color contrast and higher resolution for scaling.

This guideline applies to all situations in which images of text that are intended to be understood as text. Corporate visual guidelines beyond logo and logotype are also included in the guideline.
